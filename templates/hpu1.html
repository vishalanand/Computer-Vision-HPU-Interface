INSTRUCTIONS :

The first click CREATES / SELECTS a box and unlocks it for movement.

The next click locks the box in place

The selected box is outlined in green

Other boxes are outlined in yellow

CONTROLS :

RESIZE BOX : Mousewheel / Z / X

SELECT NEXT BOX : TAB / Click on the box

DESELECT BOX : ESC / Right click

MOVE BOX : Arrow Keys / WSAD
Note - Mouse overrides keyboard

DELETE BOX : Q

SAVE DATA : CTRL + S

Image status
Draw Boxes around the pedestrian
